# Laser cutter notes
- Epilog 120W, bed size 48x36"
- etches first, then cuts
- from laser laptop, ctrl+P, select epilog engraver, custom size
- engraving: res 500, spd 100, pwr 10
- cutting: spd 100, pwr 20, freq 50

Text labels by default are both etched (the fill) and cut (the stroke). Either will show up on the finished image, though reversibility of cutting them out is probably helpful. Also seems to be helpful to have holes in the pieces to prevent them from blowing around

My paper that's the good weight is 8.5x11, so will need to separate pieces into multiple pages. Should also help with pieces blowing around - small areas, not too bad to re-cut a specific page.
